## Docker stuff 

```bash
# make sure you are in the right folder
cd ../image_folder_dataset

# run container
docker run -it -u atom \
    -v ./:/mount_data/ \
    --name image_folder_dataset image_folder_dataset:latest bash
    
# delete container
docker rm image_folder_dataset
```
